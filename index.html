curriculum/45-day-plan.spec.ts

From your first test
to orchestrating AI agents

The whole course is one suite — seven modules, 45 days, run in this order.

  1. D 01–02

    Kickoff & setup

    The big picture of AI-first automation, and a fully working dev environment on your machine.

  2. D 03–14

    TypeScript for testers

    Types, async/await, modules and clean code patterns — the foundation every modern framework is built on.

  3. D 15–23

    Playwright, deeply

    Core test objects, codegen, locators that survive redesigns, and every kind of element interaction.

  4. D 24–32

    Debugging & reliability

    The Trace Viewer, reports and config — then kill flakiness and run fast across browsers in parallel.

  5. D 33–37

    Page objects & data-driven design

    Parameterise everything and structure a framework that scales past a hundred specs.

  6. D 38–40

    API & end-to-end journeys

    Test APIs directly and tie UI + API together into complete user journeys.

  7. D 41–45

    AI agents, MCP & CI/CD

    Agents that plan, write and heal tests via Playwright MCP — plus pipelines, sharding and your graduation project.
