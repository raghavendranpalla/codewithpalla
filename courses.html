Curriculum

Module-by-module syllabus

Eight modules, fully hands-on. Each builds on the last toward a complete AI-powered framework.

MODULE 01

TypeScript for Test Engineers

  • Types, interfaces, async/await & promises
  • Modules, npm and project setup
  • Clean, readable test code patterns
MODULE 02

Playwright Fundamentals

  • Install, config & first tests
  • Locators, auto-waiting & assertions
  • Trace Viewer, UI mode & debugging
MODULE 03

Framework Design & POM

  • Page Object Model & fixtures
  • Test data, environments & config
  • Reusable utilities & structure
MODULE 04

API, Network & Advanced

  • API testing & request context
  • Network mocking & interception
  • Auth, storage state & parallelism
MODULE 05

Playwright MCP

  • What MCP is & why it matters
  • Driving the browser via AI assistants
  • Exploring an app & capturing flows
MODULE 06

AI Test Generation & Planning

  • Turn user stories into test plans
  • Generate specs from requirements
  • Review coverage with AI
MODULE 07

Self-Healing & Agents

  • Auto-repair broken locators
  • Orchestrating multiple test agents
  • Reducing flakiness & maintenance
MODULE 08

CI/CD, Reporting & Capstone

  • GitHub Actions & sharding
  • HTML reports & artifacts
  • Capstone: ship a full framework
